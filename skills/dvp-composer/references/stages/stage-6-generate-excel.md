# 阶段 6：生成 Excel

## 职责

将验证通过的数据输出为格式化 Excel 文件。

## 输入

- `/tmp/dvp_data.json`（阶段 4 的输出，经阶段 5 验证通过）
- 用户指定的输出路径（可选）

## 做法

### 1. 确认输出路径

默认路径：`./DVP_<研究名>_<日期>.xlsx`

用户可自定义路径。

### 2. 执行生成脚本

```
python3 ${CLAUDE_SKILL_DIR}/scripts/generate_xlsx.py /tmp/dvp_data.json --output <输出路径>
```

如 openpyxl 未安装，先执行 `pip3 install openpyxl`。

### 3. 验证生成结果

确认文件成功生成，展示摘要：

```
=== 生成完成 ===
文件: ./DVP_ABC-123_2026-04-16.xlsx
章节: 8 个
总行数: 45 行
```

## 输出

`.xlsx` 文件。

## 完成条件

文件成功生成且可打开。

## 后续选项

- "调整格式" — 修改颜色、字体等
- "补充章节" — 添加新章节（回到阶段 2）
- "修正内容" — 修改特定章节（回到阶段 4）
- "重新检查" — 重新执行质量检查（回到阶段 5）
- "完成" — 结束
