# 阶段 5：全量生成

## 职责

基于确认的规则，批量生成所有章节的完整数据。

## 输入

- `/tmp/dvp_raw_materials.json`（阶段 1 的输出）
- `/tmp/dvp_template.json`（阶段 3 的输出）
- `/tmp/dvp_rules.json`（阶段 4 的输出）

## 做法

### 1. 按章节顺序逐一生成

**key-value 章节**：
- 根据 field_mapping 从原料数据填充
- 缺失字段留空，记录在缺失列表中

**table 章节**：
- 根据生成规则逐条生成数据行
- 按 generation_order 的顺序生成
- 遵循 ID 编号规则

**narrative 章节**：
- 根据内容生成指引撰写
- 参考相关文档信息

### 2. 组装完整 JSON

按 [excel-spec](../excel-spec.md) 的 schema 组装为完整 JSON。

### 3. 展示生成结果

```
=== 全量生成完成 ===

1. 研究信息      ✅ 8/8 字段
2. 数据验证范围   ✅ 1 段叙述
3. 访视计划       ✅ 7 行
4. 编辑检查       ✅ 18 行
5. 数据录入指南   ✅ 1 段叙述 + 4 行
6. Query 管理     ✅ 3 字段 + 1 段叙述
7. SDTM 映射      ⚠️ 5/8 行（3行映射逻辑待确认）
8. 签字与审批      ✅ 4 字段

准备进入质量检查。
```

## 输出

写入 `/tmp/dvp_data.json`，格式符合 [excel-spec](../excel-spec.md)。

## 完成条件

所有章节的数据生成完毕（缺失项已标注，不阻塞流程）。
