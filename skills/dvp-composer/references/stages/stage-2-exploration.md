# 阶段 2：研读与提问

## 目标

对收集到的全部资料形成充分理解，识别出编写 DVP 所需的确定性知识和待确认的疑问。

## 动作

1. 通读全部已收集资料，逐份理解其内容与用途。
2. 梳理资料间的关联，建立对研究项目的整体认知。
3. 识别编写 DVP 所需但资料中未明确的信息，逐条列出待确认问题。
4. 对每个问题尝试给出初步判断或假设，以便与团队确认时有的放矢。
5. 将问题按优先级排序，区分阻塞项和可延后确认项。

## 输入

- `/tmp/dvp_raw_materials.json`（阶段 1 的输出）

## 输出

更新 `/tmp/dvp_raw_materials.json`，追加探索结果：

```json
{
  "exploration": {
    "study_overview": "研究整体概貌描述",
    "questions": [
      {
        "topic": "问题主题",
        "question": "具体问题",
        "hypothesis": "初步判断或假设",
        "priority": "blocking | deferrable",
        "resolution": "用户回答或待定"
      }
    ]
  }
}
```

## 完成条件

- 已通读全部资料并形成整体认知
- 待确认问题已逐条列出并附带初步假设
- 阻塞项已向用户提问并获得回应，或用户标记为稍后处理

## 回退

用户补充新资料时，回到阶段 1。在阶段 1 中更新资料后重新进入本阶段。
