# Dvp Composer

`Dvp-Composer` is a Claude Code Plugin for **Data Managers** at CROs (Contract Research Organizations), helping them compose **DVPs (Data Validation Plans)** for clinical trials.

## Commit Convention

Every git commit must include this trailer at the end of the message:

```
Co-Authored-By: GLM 5.1 <noreply@z.ai>
```
